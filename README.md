# Algorithm Quest

Algorithm Quest is an educational game designed using JavaFX, SceneBuilder, Multithreading, and Socket Programming to help students learn computer algorithms through engaging, real-life-based storylines. Each level represents an algorithm visualized through problem-solving missions. Players explore environments, solve puzzles, and compete to reach goals using algorithmic strategies.

## Features

- Real-time multiplayer
- Algorithm visualization
- Timer countdown synchronization
- Individual learning experience

## Getting Started

1. Clone the repository to your local machine
2. Import the project into your preferred Java IDE
3. Run it on your local machine

## Prerequisites

- Java 21 or higher
- VS Code
